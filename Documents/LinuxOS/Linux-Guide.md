User Management in Linux
---------------------------
Linux has 3 types of Users
Normal User
root User
Super User

Command to create a new user in Linux is:
useradd or add user

Command to create a new group is:
addgroup

Command to add previliges as root does:
sudo

Localhost=(ALL:ALL)
